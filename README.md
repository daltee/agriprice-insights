# AgriPrice

Real-time crop and produce price tracking for East African markets.

## Tech Stack

- TanStack Start (React 19, Vite 7)
- Tailwind CSS v4 + shadcn/ui
- Lovable Cloud (Postgres, Auth, Storage)

## Getting Started

```bash
bun install
bun run dev
```

Build for production:

```bash
bun run build
```
